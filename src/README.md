# 📄 Zarvix Builder

Zarvix Builder is a premium, highly responsive, and ATS-optimized Resume Builder application. It allows users to input their professional details and generate a pixel-perfect, print-ready PDF resume in real-time.

### 🌐 Live Demo
[Zarvix Builder - Live on Vercel](https://my-first-project-three-tau.vercel.app/)

## ✨ Key Features
* **Real-time Live Preview:** See your resume update instantly as you type.
* **Pixel-Perfect PDF Export:** Custom `@media print` CSS ensures no blank pages or broken layouts when downloading.
* **Modern Dark-Mode UI:** Built with a sleek, SaaS-like premium dark theme.
* **Form Validation:** Prevents incomplete resume downloads with smart alerts.
* **Custom AJAX Contact Form:** Seamless form submission using FormSubmit with a beautiful custom popup (no page reloads).

## 🚀 Tech Stack
* **Frontend:** React.js, Vite
* **Styling:** Tailwind CSS, Framer Motion
* **Deployment:** Vercel
